Notes
-----

.. note::
    - Requires the bigsuds Python package on the remote host. This is as easy as pip install bigsuds


Examples
--------

.. code-block:: yaml

    
    - name: Create a view foo.local
      local_action:
          module: "bigip_view"
          user: "admin"
          password: "admin"
          name: "foo.local"

    - name: Assign zone "bar" to view "foo.local"
      local_action:
          module: "bigip_view"
          user: "admin"
          password: "admin"
          name: "foo.local"
          zones:
              - "bar"
